IP - указатель на очередную команду (в кодовом сегменте)
Базовый адрес (добавить 0 в конце) + IP = физический адрес
489D + 3 = 489D3

В сегменте данных он ищет по названию переменной
В сегменте стека указатель SP (stack pointer), на вершину стэка

Пример 1. (Часть 2)

db 256 dup (?) = резервирование 256 байт
может вместо dp быть dw
знак вопроса - заполняем НИЧЕМ

Hello db 'Hello, World!$'
Hello - название переменной, db - 1 байт, $ - конец строки, иначе будет мусор
