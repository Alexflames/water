Sp = Sp(n) - возрастающая

g = t(n) / (t(n) + p(n) / p) - закон Густафсона-Барсиса,

где t(n) - время последовательного выполнения, 
    p(n) - время параллельной части вычислений

T1 = t(n) + p(n)
Tp = t(n) + p(n) / p

t(n) = g(t(n) + p(n)/p)
p(n) = (1-g)*p*(t(n)+p(n)/p)

Sp = T1/Tp = (t(n) + p(n)) / (t(n) + p(n)/p) =
= (t(n) + p(n)/p) * (g + (1-g) * p) / (t(n) + p(n)/p) =
= g + (1 - g) * p = p + (1-p) * g

Если доля последовательных вычислений 0 то g
                                      1 то 1
Другое название - ускорение масштабирования - scale speedup

5. Двоичный ГИПЕРКУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУУБ

В n-мерном пространстве имеются точки (x1, x2, ..., xn) xi - либо 0 либо 1
Получается n-мерный куб
  N = 2^n - число точек

      (1, 1)              n = 3  -> куб, грани которого - ромбы
        /\
       /  \
(0,1) /    \ (1, 0)
      \    /
       \  /
        \/
       (0,0)

Свойство:
1. 2 процессора имеют соединение если двоичное представление их номеров имеет 
только одну различающуюся позицию
2. Каждый процессор связан с log(2)(N) непосредственными соседями
3. Кратчайший путь между 2 любыми процессорами имеет длину совпадающую с 
количеством различающихся битовых значений - расстояние Хэмминга

Пример - Cosmic cube (1983)
