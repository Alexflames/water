<==========>
Машина поста.
Тезис - "Всякий алгоритм представим в форме машины Поста"

Алгоритм - программа для машины Поста, приводящая к решению поставленной задачи.
Если задача имеет алгоритмическое решение, то она представима в форме команд для машины Поста

[ ][ ][ ][V][V][ ][ ][V][ ][ ]
                ^

-> Шаг вправо   X стереть отметку
<- Шаг влево    
?a: b посмотреть ячейку, если 0, то перейти в ячейку с номером a, иначе на команду с номером b
V записать отметку 
! остановка

Остановка по команде стоп (!) - корректность алгоритма, результативность
Остановка при недопустимости команды, запись метки где есть / стереть где нет
Машина не останавливается никогда, уход в бесконечность

Справа на лево двигаемся в ленте. Если ячейка пустая, то стоп, иначе возвращаемся к 1
1 <- 2
2 ? 3; 1
3!

С помощью 1 и 2 двигаемся до не пустой
1->2
2? 1;3
3<-4
4 V 5
5!

На ленте машины поста массив из n меток. Проверка делимости числа n на 3. Если да, то через 1 пустую ячейку поставить метку
1->2  Идём вправо
2?3;4 Пустая ячейка? Переход на 3
3!    
4->5  Идём вправо
5?3;6 Пустая? Всё плохо
6->7  Снова вправо
7?8;1 Пустая? 
8->9  Идём вправо
9V3   Ставим метку

Зацикливание
1->2
2<-1

В опросе будет задание про кратность, суммирование 2 чисел, увеличение числа с помощью машины Поста

Машина Тьюринга

Формальное описание машины Тьюринга:
--------------------------------------|
Конечное мн-во состояний Q            |
Конечное мн-во символов ленты A       |
Функция переходов дек.произв. Q x A   |
Пустой символ e                       |
Подмножество E < A                    |
Одно из состояний называется начальным|
--------------------------------------|

Способ задания МТ
1. Список команд
A = {a0 = la1 = 1} и Q = {q0, q1, q2}
q1'lambda' -> q21E
q11 -> q11R
q21 -> q21L
q2'lambda' -> q0'lambda'R

2. Таблица переходов
E -> остаться на месте L -> пойти влево R-> пойти вправо

3. Граф переходов

Приведение конфигураций к стандартному виду - когда обозреваем первую букву нашего слова
Стандартная начальная / заключительная конфигурация
Чтобы слово - результат можно было передать для обработки другой машине тьюринга
