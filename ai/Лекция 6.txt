	Нечеткая логика

A*, A->B => B*
Пусть есть функция y = f(x)
Пусть на Ox есть точка а,
	  на Оу есть точка b
b = f(a)   или   a -> b    или     a -> f(a)

Теперь возьмем посложнее!
Пусть есть интервальная функция F(x), значение в каждой точки - интервал (между 2 функциями)
					  _
На Ox отметим a1,a2 = a

Найдем их проекцию на Оy
b = F(a)

Композиционное правило вывода (Заде)          | Заде - ученый такой
Им предложено заменить первую запятую на композицию
A*, A -> B => B*
   |
   v
B = A o F

	Нечеткие экспертные системы
L_1 если A_11 и/или А_12 и/или ... и/или A_1n то B_11 и/или ... B_1m 
L_2 если A_21 и/или А_22 и/или ... и/или A_2n то B_21 и/или ... B_2m
...
L_k если A_k1 и/или А_k2 и/или ... и/или A_kn то B_k1 и/или ... B_km  

L1 если А1 то B1
................
Lk если Аk то Bk

Это большой перебор, поэтому обычно одна переменная выхода (B_k1)

Этапы (чего?)
1. Фаззификация  (Fuzzy - нечеткий)
С помощью функции принадлежности всех термов входных лингвистических
переменных и на основании задаваемых четких значений из универсумов
входных лингвистических переменных определяется степень уверенности что
соответствующая лигвистическая переменная принимает конкретное значение
(То есть конкретное значение, например, 20 градусов, превращаются
 вероятность того или иного лингвистического вывода температуры)

2. Этап непосредственного логического вывода
	1) На основании набора правил нечеткой базы знаний вычисляется значение 
	истинности для предпосылки каждого правила

	2) Используя один из способов построения нечеткой импликации получим
	нечеткую переменную, соответствующую значению выходной 
	лингвистической переменной (выбор операции какой-то такой этой вот короче)

3. Композиции (агрегации, аккумуляции)
ВСЕ нечеткие множества, назначенные (вычисленные) для каждого терма каждой
выходной лингвистической переменной объединяются вместе и формируется единое 
нечеткое множество - значение для каждой выходной лингвистической переменной
sum, max аккумуляции

[4. Дефаззификация]
Используется при необходимости преобразовать нечеткий набор значений
выходных лингвистических переменных в четкий (набор конкретных функций)
Существует порядка 30 методик подобной операции
	1. Center Of Gravity (COG)
	2. Mean Of Maximums (MOM)   
	3. First Maximum

Типы шкал
1. Шкала наименований или классификаций
2. Шкала порядка применяется для измерения упорядочивания объектов
	по одному или нескольким признакам. Задается только порядок
	следования, но невозможно сказать насколько лучше другого
3. Шкала рангов - шкала порядка со значениями 1, 2, ...
4. Шкала интервалов - применяется для отображения величины 
	различия между свойствами объектов, может иметь произвольный
	масштаб и точки отсчета
5. Шкала отношений - во сколько раз больше
6. Шкала разностей - частный случай шкалы интервалов при выборе 
единицы масштаба 
7. Абсолютная шкала - фиксирован масштаб и точка отсчета

Методы измерений
1. Ранжирование - эксперт располагает объекты в порядке предпочтения,
	руководствуясь одним или несколькими ... сравнения
2. Парная оценка - представляет собой процедуру установления предпочтения 
	объектов при сравнении всех возможных пар
3. Непосредственная оценка - объектам приписываются числовые значения по шкале интервалов

Методы построения функции принадлежности
	1.Прямые и 2.косвенные
1. В случае прямого метода эксперт непосредственно задает правила определения
значений функции принадлежности, характеризующие данные понятия

Для любых u1,u2 из U мю_А(u1) < мю_А(u2)

2. Эксперт выдает значение, удовлетворяющее заранее сформулированным условиям, а 
в дальнейшем эта информация проходит дополнительную обработку.

Метод семантических дифференциалов
1. Определить список свойств, по которым оценивается понятие/объект
2. Найти в этом списке полярные свойства и сформировать полярную шкалу
3. Для каждой пары полюсов определить в какой степени оцениваемые понятия
	обладают положительным свойством

Например: высота лба	низкий-широкий
		  профиль носа  горбатый-курносый
		  длина носа 	короткий-длинный
		  разрез глаз   извините рамзан ахматович
		  цвет глаз 	темный-светлый

Пусть есть покрытие К { A_1, ..., A_k } и A_i =/= 0 (не пустое мн-во)

Тогда для некоторого B из u
мю_B(Ai) = |A_i и B| / |A_i или B|

Пусть u = {1..9}
		  A1 		A2 		 A3 	  A4 	   A5
K = { {1,2,3,5}, {3,6,9}, {2,4,8}, {1,3,7}, {2,3,8} }
B = {2,3,5,8,9}

Вычисляем:
мю_B = {<A1, 1/2>, <A2, 1/3>, <A3, 1/3>, <A4, 1/7>, <A5, 3/5>}
