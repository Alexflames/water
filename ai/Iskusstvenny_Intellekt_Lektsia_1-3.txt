Artificial Intelligence
Цели и задачи ИИ:
1) Информационная (или Эвристическая, или Прагматическая) - состоит в создании программ для вычислительных машин, для автоматизации таких видов человеческой деят-ти, которые традиционно считаются интеллектуальными. При этом, основное значение имеет лишь конечный результат, а именно, насколько он совпадает с таким же результатом человека (той же задачи, алгоритм решения не важен, главное - результат).
2) Бионическая - состоит в использовании программ ИИ для объяснения умственной деят-ти человека во время решения им тех или иных задач.

	Этапы развития "Машинного Интеллекта"
I этап: 50-е - 60-е гг.
II этап: 60-е - 70-е гг. - период активного формирования Баз Данных (исп-ся и сортируются большие объемы информации)
III этап: 70-е - 80-е гг. - появление класса задач, характеризующихся отсутствием заранее заданных схем решений (так появилось Эвристическое программирование)
IV этап: 80-е - 90-е гг. - выделение категории "ЗНАНИЯ" для синтеза алгоритмов решения задач. Активное развитие Экспертных систем (или систем, основанных на знаниях)
V этап: 90-е - ~ гг. - усложнение систем связи решаемых задач потребовало качественно нового уровня интеллектуальности программных систем, таких как защита от несанкционированного доступа, информационная безопасность ресурсов, информационный поиск и анализ информации и т.д.

	Подходы к проектированию систем ИИ
1. Логический подход
2. Структурный подход (один из самых ярких примеров - Нейронные сети)
3. Эволюционный подход (генетические алгоритмы)
4. Имитационный подход
5. Интегральные роботы

Есть еще пустые мн-ва
Операции над мн-вами: объединение, пересечение, дополнение (эти три самые важные, фундаментальные)
A U B = {x|x принадлежит A v x принадлежит B}
(пересечение - тоже самое, только ИЛИ на И заменяется)
НЕ A = {x принадлежит U, x НЕ принадлежит A}
A c (включается в) B = x принадлежит A --> x принадлежит B

Свойства:
(объединение будем обозначать как u, пересечение - как n)
A u НЕ A = U
A n НЕ A = 0
A u/n B = B u/n A (коммутативность)
(Дистрибутивность:)
A u (B n C) = (A u B) n (A u C)
A n (B u C) = (A n B) u (A n C)
A u/n (B u/n C) = (A u/n B) u/n C (ассоциативность)
(Идемпотентность:)
A u A = A
A n A = A

Мощность - количество элементов для конечных множеств.

Характеристический вектор - двоичный вектор, 0 - эл-та в мн-ве нет, 1 - есть.
Пример:
U = {0, 1, 2, 3, 4, 5}
A = {0, 2, 4}
X = (1, 0, 1, 0, 1, 0)

ЧЕТКИЕ ОТНОШЕНИЯ
Отношением на множествах X1,X2,...,Xn наз-ся любое подмножество декартового произведения этих множеств (R входит в X1 x X2 x..x Xn)

R входит в AxB или R э (a,b) (или aRb)
Способы задания отношений:
1. Табличный
2. Графовый

табличный:
  1 2 3 4
1 0 1 0 0
2 1 0 1 1
3 0 1 0 1
4 0 1 1 0

графовый:
2----3
|\   |
| \  |
1   4

Свойства отношений:
Рефлексивность (для любого x принадл A пара R э (x,x))
Антирефлексивность
Симметричность (для любых x,y принадл А пара R э (x,y) -> R э (y,x))
Антисимметричность
Транзитивность (для любых x,y,z принадл A вып-ся: если R э (x,y) и R э (y,z), то R э (x,z))

Классы:
отношение эквивалентности (рефлексивность, симметричность, транзитивность)
отношение порядка (рефлексивность, антисимметричность, транзитивность)
отношение толерантности (симметричность, рефлексивность)

	Обратное отношение R^(-1)(x,y) = R(y,x)
Свойства:
1) Рефлексивность 
  E входит в R (или для любого x R(x,x) = 1)
2) Сильная рефлексивность
 для любых x,y R(x,y) < 1
3) Слабая рефлексивность
 для любых x,y R(x,y) <= R(x,x)
4) Антирефлексивность
 RnE = 0 (или для любого x R(x,x) = 0)
5) Сильная антирефлексивность
 для любых x,y R(x,y) > 0
6) Слабая антирефлексивность
 для любых x,y R(x,y) >= R(x,x)
7) Симметричность
 R = R^(-1) (или для любого x R(x,y) R(x,y) = R(y,x))
 8) RnR^(-1) = E - антисимметриность
 9) RnR^(-1) = 0 - асиметричность
 10) Сильная линейность
